Drawing a Scatterplot with d3

Your scatterplot already shows the distribution of the data values with respect to the data domain. However, without any labels it is very hard to actually derive individual values from it. d3 is a JavaScript library "for manipulating documents based on data". It provides a lot of functionality for creating interactive visualizations for the web.

Have a look at the d3 introduction to understand its principles. Most importantly, d3 does not really create visualizations by itself but allows you to bind data to elements in the DOM and set their attributes based on that data. While d3 can also use the canvas, it is mainly used together with Scalable Vector Graphics (SVG). Make sure to have a basic understanding of how SVG works (e.g. by reading this documentation).

In our example the data is the array of movies. Every individual movie is tied to a circle and we want to set its 2d position based on the corresponding attribute values. d3 provides scales that help with the mapping from data values to visual attributes.
